    # Markdown

    Je to spôsob zapisovania textu, ktorý sa následovne dá prekladať do napr. HTML
    Markdown súbory sa končia .md

    # Prečo ho používať

    2. Používajú ho programátori
    3. Čítateľjenší než HTML
